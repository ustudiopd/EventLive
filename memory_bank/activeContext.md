# 현재 작업 상황 (Active Context)

## 1. 현재 집중하고 있는 작업  
- **작업명**: 웨비나 시청 페이지 UI/UX 개선 및 4K 모니터 지원
- **목표**: 
  - 4K 모니터에서 레이아웃이 적절하게 표시되도록 최적화
  - 채팅 패널이 일정 비율 이상 늘어나지 않도록 제한
  - 전체화면 기능이 브라우저 네이티브 API로 정상 작동하도록 구현
- **상태**: ✅ 완료
  - 전체 너비 레이아웃 적용
  - 채팅 패널 최대 너비 400px 제한
  - 브라우저 네이티브 Fullscreen API 구현
  - 모바일 반응형 레이아웃 (영상 → 참여자 → 채팅 순서)
  - 세션 소개 섹션 추가 (리액션 패널 대체)

## 2. 다음 예정 작업  
- **우선순위 높음**: 
  1. 설문조사 기능 구현 (새로 요청됨)
     - 설문조사 생성/관리 (운영 콘솔)
     - 설문조사 응답 (참여자)
     - 설문조사 결과 조회/통계
  2. 발표자료 다운로드 기능 구현 (새로 요청됨)
     - 파일 업로드 (웨비나 생성/수정 시)
     - 파일 다운로드 (참여자)
     - 파일 관리 (운영 콘솔)
  3. 퀴즈 기능 구현
  4. 추첨 기능 구현

- **우선순위 중간**:
  5. 웨비나 등록 페이지
  6. 게스트 모드

- **우선순위 낮음**:
  7. 리포트 고도화
  8. Typing 표시 기능 완성
